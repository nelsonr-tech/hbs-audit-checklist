🔵 Phase 1 — Asset Discovery
Tool: Nmap / Zenmap (Free) · ~30–60 min
0 / 0
📥 Download: nmap.org/download
  • Download and install Nmap / ZenmapZenmap is the GUI version included in the Windows installer.
  • Run host discovery sweep on all subnetsnmap -sn 192.168.x.0/24
  • Run detailed OS + service scannmap -A -T4 192.168.x.0/24
  • Export scan results to XMLnmap -A -oX audit_scan.xml 192.168.x.0/24
  • Compare discovered hosts against DHCP lease exportUnknown hosts = rogue devices. Flag immediately.
  • Flag devices running Telnet, FTP, or plain HTTPPort 23, 21, 80 — all transmit credentials in cleartext.
  • Flag devices on unusual or unexpected portsAny unrecognized service = investigate before proceeding.
  • Build or update network asset inventorySpreadsheet: IP, MAC, hostname, OS, open ports, location, owner.
